units.getQuantity(quantity, recognized_units)[source]¶

Split a string ‘Value Units’ into numerical value and string-units, and create an instance of astropy.units.Quantity.

Parameters:
  • quantity (str) – E.g. quantity='1 Jy' would return <Quantity 1.0 Jy>
  • recognized_units (tuple) – Tuple of valid units (as strings) for quantity. E.g., for a length quantity, recognized units could be (‘micron’,’m’,’pc’,’AU’), etc.
Returns:

quantity – Instance of class astropy.units.Quantity, i.e. a value with units.
Return type:

instance

Examples

getQuantity('1 m',('micron','m','pc','AU'))
  <Quantity 1.0 m>

getQuantity('1 W',('micron','m','pc','AU'))
  ValueError: Specified unit 'W' is not recognized. Recognized are: m,AU,pc
units.getValueUnit(quantity, recognized_units)[source]¶

Split string into value and units string.

Evaluate both and return the numerical value, and the units object.

Parameters:
  • quantity (str or Quantity instance) –

    If string, its format is num units, where num is the string representation of a numerical value, and units is one of the recognized units in the sequence of strings provided in recognized_units. num and units can but need not be separated by whitespace.

    If instance of Quantity, quantity.value and quantity.unit are the equivalents of num and unit.

  • recognized_units (seq of strings) – Sequence of strings representing units that the units part of string will be checked against.

Examples

recognized_units = ('Jy/arcsec^2','mJy/mas^2','uJy/sr')
getValueUnit('5.2 Jy/arcsec^2',recognized_units)
  (5.2, Unit("Jy / arcsec2"))
